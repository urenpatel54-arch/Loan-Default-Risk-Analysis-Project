# Loan Default Risk Analysis Project

## Overview

This project is based on Statistics, Probability and Linear Algebra concepts using Python programming. The main objective of this project is to analyze customer loan default risk using statistical calculations, probability analysis and data visualization techniques.

## Objectives

* Analyze customer loan data
* Calculate mean, median and mode
* Find variance and standard deviation
* Calculate probability and conditional probability
* Apply linear algebra concepts such as dot product, norm and angle between vectors
* Create graphs and visualizations for better understanding

## Dataset Fields

* Customer_ID
* Age
* Income
* Loan_Amount
* Credit_Score
* Loan_Term
* Default_Status

## Tools and Libraries Used

* Python
* Pandas
* NumPy
* Matplotlib
* SciPy
* VS Code

## Graphs Included

* Histogram
* Q-Q Plot
* Scatter Plot
* Heatmap
* Boxplot
* Pie Chart

## Final Insights

1. Customers with low credit scores have higher default risk.
2. Customers with higher income usually take larger loan amounts.
3. Probability and statistics help banks identify risky customers.
4. Correlation analysis helps understand relationships between variables.
5. Linear algebra concepts help represent financial data mathematically.

## Conclusion

This project helped in understanding how statistics, probability and linear algebra can be used in real-world financial data analysis. Python programming and data visualization techniques made the analysis easier and more understandable.
